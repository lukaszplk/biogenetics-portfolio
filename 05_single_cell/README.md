# Single-Cell RNA-seq Analysis

_Coming soon._

This project will demonstrate:
- Preprocessing scRNA-seq count matrices (filtering, normalization, HVG selection)
- Dimensionality reduction: PCA → UMAP
- Leiden community detection clustering
- Cell-type marker visualization (dot plots, violin plots)

## Planned Tools

- `scanpy` — single-cell analysis framework
- `anndata` — annotated data matrices
- `umap-learn` — UMAP embedding
- `leidenalg` — graph-based clustering
